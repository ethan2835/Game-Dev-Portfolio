# Console Card Dealer

A simple C# console application that shuffles a deck and deals cards to 4 players.

## Features
- Shuffles a standard 52-card deck
- Deals 3 cards to each of 4 players
- Displays cards in readable format (e.g., "Ace of Spades")
- Determines and displays the player with the highest card

## How to Run
1. Open the solution in Visual Studio
2. Build and run the project
3. Type any key (except 'q') to deal a new round
4. Type 'q' to quit

## Technologies
- C#
- ConsoleCards library (provided by course)

## Learning Outcomes
- Working with classes and objects (`Deck`, `Card`)
- Enum handling for card ranks
- Basic game logic and winner detection
- Console I/O and string formatting
